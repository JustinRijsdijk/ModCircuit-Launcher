# Wails Nuxt Tailwind Template

A minimal template to get you started building desktop applications with [Wails 2](https://wails.io), [Nuxt 4](https://nuxt.com), [Nuxt Tailwind](https://tailwindcss.nuxtjs.org/), and [Nuxt Color Mode](https://color-mode.nuxtjs.org/).

## Quick Start

Create a new project using this template:

```bash
wails init -n "Your Project Name" -t https://github.com/paulbrickwell/wails-nuxt-tailwind-template
```

Navigate to your project and start developing:

```bash
cd "Your Project Name"
wails dev
```

## What's Included

This template provides the essentials plus a few extras for building a Wails desktop app with Nuxt:

- Wails 2
- Nuxt 4
- Nuxt Tailwind Module
- Nuxt Color Mode Module

## Basic Version

Looking for a basic template without the Nuxt Tailwind and Nuxt Color Mode modules? Check out the [wails-nuxt-template](https://github.com/paulbrickwell/wails-nuxt-template) for a basic template.
